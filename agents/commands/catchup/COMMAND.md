# Catchup Command

## Purpose

Quickly understand what has changed in the current working branch compared to main/master.

## Usage

```
/catchup
```

## Behavior

- Compares current branch against main/master
- Reviews commit messages for context
- Reads all changed files
- Summarizes completed work and current state
- Identifies potential issues or blockers
